# HStack

HStack arranges its child elements horizontally, making it a suitable layout component for creating a row of items. It is worth noting that Row is just an alternative name for HStack.

## Usage

<UsageWrap>
  <Center>
    <HStack space="sm">
      <Box w={100} h={100} bg="$cyan300" />
      <Box w={100} h={100} bg="$cyan400" />
      <Box w={100} h={100} bg="$cyan500" />
      <Box w={100} h={100} bg="$cyan600" />
    </HStack>
  </Center>
</UsageWrap>

```jsx
import { HStack, Box } from '@utilitywarehouse/native-ui';

const MyComponent = () => (
  <HStack space="sm">
    <Box w={100} h={100} bg="$cyan300" />
    <Box w={100} h={100} bg="$cyan400" />
    <Box w={100} h={100} bg="$cyan500" />
    <Box w={100} h={100} bg="$cyan600" />
  </HStack>
);
```

## Props

| Property | Type                                                     | Description                                                 | Default |
| -------- | -------------------------------------------------------- | ----------------------------------------------------------- | ------- |
| space    | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl'` | The space between the HStack items.                         | -       |
| reversed | `boolean`                                                | When true, it places the HStack items in reverse direction. | -       |
| wrap     | `boolean`                                                | When true, it wraps the children to the next line.          | -       |
