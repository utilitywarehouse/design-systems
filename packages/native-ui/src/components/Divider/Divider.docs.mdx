# Divider

Divider components in `native-ui` enhance visual organisation by creating clear boundaries between sections, improving readability and content grouping. They reinforce the visual hierarchy, making it easier for users to understand relationships between elements.

## Usage

<UsageWrap>
  <Center>
    <Text>This text is divided</Text>
    <Divider space="sm" />
    <Text>From this text</Text>
  </Center>
</UsageWrap>

```tsx
import { Spinner, Center } from '@utilitywarehouse/native-ui';

const MyComponent = () => (
  <Center>
    <Text>This text is divided</Text>
    <Divider space="sm" />
    <Text>From this text</Text>
  </Center>
);
```

## Props

| Property | Type                                                                       | Description                          | Default |
| -------- | -------------------------------------------------------------------------- | ------------------------------------ | ------- |
| `color`  | `string`                                                                   | The color of the spinner.            | `-`     |
| `space`  | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl'\| '4xl'` | The space between the Divider items. | `none`  |
