## Usage

> This component should be wrapped in a `NativeUIProvider` component to ensure that the correct theme is applied.

Badge is a stateless component, e.g. it has no Pressed, Disabled, Active states. Badges should not be used as a mechanism for:

- Filtering content
- Assigning a piece of content to a group (tagging)
- Performing an action

<UsageWrap>
  <Center>
    <Badge colorScheme="green">Ready</Badge>
  </Center>
</UsageWrap>

```tsx
import { Badge } from '@utilitywarehouse/native-ui';

const MyComponent = () => <Badge colorScheme="green">Ready</Badge>;
```

## Props

| Property      | Type                 | Description                                                | Default |
| ------------- | -------------------- | ---------------------------------------------------------- | ------- |
| `colorScheme` | `string`             | The color scheme to use for the badge.                     | `cyan`  |
| `strong`      | `boolean`            | Whether to use a strong color scheme.                      | `false` |
| `borderless`  | `boolean`            | Whether to remove the bottom border radius from the badge. | `false` |
| `size`        | `'small' \| 'large'` | The size of the badge.                                     | `md`    |

## Advanced Usage

Although not recommended, the `Badge` component can be used to create more complex UI with the internal parts of the badge.

<UsageWrap>
  <Center>
    <Badge colorScheme="cyan">
      <BadgeText>Add</BadgeText>
    </Badge>
  </Center>
</UsageWrap>

```tsx
import { Badge, BadgeText } from '@utilitywarehouse/native-ui';

const MyComponent = () => (
  <Badge colorScheme="cyan">
    <BadgeText>Add</BadgeText>
  </Badge>
);
```
