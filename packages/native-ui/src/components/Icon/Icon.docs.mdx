## Usage

You can either use the React Native components directy from our `@utilitywarehouse/react-native-icons` package or use the `Icon`
component from our `@utilitywarehouse/native-ui` package to render the icons with utility props such as `color`.

The following example shows how to use the icons directly from the `@utilitywarehouse/react-native-icons` package.

<UsageWrap>
  <Center>
    <HStack space="lg">
      <ElectricityMediumIcon color={colorsCommon.serviceElectricity} />
      <MobileMediumIcon color={colorsCommon.serviceMobile} />
      <BroadbandMediumIcon color={colorsCommon.serviceLandline} />
      <InsuranceMediumIcon color={colorsCommon.serviceInsurance} />
    </HStack>
  </Center>
</UsageWrap>

```jsx
import { HStack, Icon, color } from '@utilitywarehouse/native-ui';
import {
  ElectricityMediumIcon,
  MobileMediumIcon,
  BroadbandMediumIcon,
  InsuranceMediumIcon,
} from '@utilitywarehouse/react-native-icons';

const MyComponent = () => {
  return (
    <HStack space="lg">
      <ElectricityMediumIcon color="$serviceElectricity" />
      <MobileMediumIcon color="$serviceMobile" />
      <BroadbandMediumIcon color="$serviceLandline" />
      <InsuranceMediumIcon color="$serviceInsurance" />
    </HStack>
  );
};
```

Or you can use the `Icon` component to render the icons.

```jsx
import { HStack, Icon, color } from '@utilitywarehouse/native-ui';
import {
  ElectricityMediumIcon,
  MobileMediumIcon,
  BroadbandMediumIcon,
  InsuranceMediumIcon,
} from '@utilitywarehouse/react-native-icons';

const MyComponent = () => {
  return (
    <HStack space="lg">
      <Icon as={ElectricityMediumIcon} color="$serviceElectricity" />
      <Icon as={MobileMediumIcon} color="$serviceMobile" />
      <Icon as={BroadbandMediumIcon} color="$serviceLandline" />
      <Icon as={InsuranceMediumIcon} color="$serviceInsurance" />
    </HStack>
  );
};
```

## Props

| Name    | Type                  | Description                   | Default |
| ------- | --------------------- | ----------------------------- | ------- |
| `as`    | `React.ComponentType` | The icon component to render. | -       |
| `color` | `string`              | The color of the icon.        | -       |
