## React Native Aria

[React Native Aria](https://react-native-aria.geekyants.com/) brings the support of React Aria to React Native (Web, Android & iOS).

## `useTheme`

`useTheme` is a custom hook which return the current theme.

```tsx
import { useTheme } from '@utilitywarehouse/native-ui';
```

Let's use our `useTheme` value to give background color to `View` from `react-native`.

```tsx
import { View } from 'react-native';
import { useTheme } from '@utilitywarehouse/native-ui';

function Example() {
  const theme = useTheme();
  return (
    <View
      style={{
        width: 100,
        height: 100,
        backgroundColor: theme.colors.green500,
      }}
    />
  );
}
```

## `useBreakpointValue`

`useBreakpointValue` is a custom hook to returns the value for the current breakpoint based on the provided responsive values object. It is also responsive to window resizing and returning the appropriate value according to the new window size.

```tsx
import { useBreakpointValue } from '@utilitywarehouse/native-ui';
```

```tsx
const BreakPointValue = () => {
  const flexDir = useBreakpointValue({
    base: 'column',
    sm: 'row',
  });

  return (
    <View
      style={{
        flexDirection: flexDir,
        gap: 10,
      }}
    >
      <StyledBox>
        <StyledText>Universal</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Performant</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Accessible</StyledText>
      </StyledBox>
    </View>
  );
};
```

## `useMedia`

`useMedia` is a custom hook that returns the breakpoints object with boolean value if your current screen size is a particular breakpoint or not. It is also responsive to window resizing and returning the appropriate value according to the new window size.

```tsx
import { useMedia } from '@utilitywarehouse/native-ui';
```

```tsx
const MediaExample = () => {
  const media = useMedia();

  return (
    <View
      style={{
        flexDirection: media.lg ? 'row' : 'column',
        gap: 10,
      }}
    >
      <StyledBox>
        <StyledText>Universal</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Performant</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Accessible</StyledText>
      </StyledBox>
    </View>
  );
};
```

## `useColorMode`

`useColorMode` is a custom hook which return current color mode.

```tsx
import { useColorMode } from '@utilitywarehouse/native-ui';
```

Let's use our `useColorMode` value to give background color to `View` from `react-native`.

```tsx
import { View } from 'react-native';
import { useColorMode } from '@utilitywarehouse/native-ui';

function Example() {
  const [colorMode, setColorMode] = useColorMode();
  return (
    <View
      style={{
        width: 100,
        height: 100,
        backgroundColor: colorMode === 'light' ? 'white' : 'black',
      }}
    />
  );
}
```

## `useLightDark`

`useLightDark` is a custom hook which return current color mode as `light` or `dark`.

```tsx
import { useLightDark } from '@utilitywarehouse/native-ui';
```

Let's use our `useLightDark` value to give background color to `View` from `react-native` based on light and dark mode.

If the current color mode is `light`, then the background color will be `white`, otherwise `black`.

```tsx
import { View } from 'react-native';
import { useLightDark } from '@utilitywarehouse/native-ui';

function Example() {
  const viewBg = useLightDark('white', 'black');
  return (
    <View
      style={{
        width: 100,
        height: 100,
        backgroundColor: viewBg,
      }}
    />
  );
}
```
