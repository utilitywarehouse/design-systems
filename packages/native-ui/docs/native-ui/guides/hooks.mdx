# Hooks

The following hooks are available in the Native UI library:

- [React Native Aria](#react-native-aria)
- [`useStyles`](#useStyles)
- [`useToken`](#useToken)
- [`useBreakpointValue`](#useBreakpointValue)
- [`useMedia`](#useMedia)
- [`useColorMode`](#useColorMode)

## React Native Aria

[React Native Aria](https://react-native-aria.geekyants.com/) brings the support of React Aria to React Native (Web, Android & iOS).

## `useStyles`

`useStyles` is a hook to resolve values from the theme, the current breakpoint value and the styles to pass to components.
[Read more here](/?path=/docs/native-ui-guides-styling-usestyles--docs)

```tsx
import { useStyles } from '../../';

const { theme, breakpoint } = useStyles();
```

## `useToken`

`useToken` is a custom hook to resolves design tokens from the theme.

```tsx
import { useToken } from '../../';
```

Let's use our `useToken` value to give background color to `View` from `react-native`.

```tsx
import { View } from 'react-native';
import { useToken } from '../../';

function Example() {
  const resolvedGreen = useToken('colors', 'green500');
  return (
    <View
      style={{
        width: 100,
        height: 100,
        backgroundColor: resolvedGreen,
      }}
    />
  );
}
```

## `useBreakpointValue`

`useBreakpointValue` is a custom hook to returns the value for the current breakpoint based on the provided responsive values object. It is also responsive to window resizing and returning the appropriate value according to the new window size.

```tsx
import { useBreakpointValue } from '../../';
```

```tsx
const BreakPointValue = () => {
  const flexDir = useBreakpointValue({
    base: 'column',
    sm: 'row',
  });

  return (
    <View
      style={{
        flexDirection: flexDir,
        gap: 10,
      }}
    >
      <StyledBox>
        <StyledText>Universal</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Performant</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Accessible</StyledText>
      </StyledBox>
    </View>
  );
};
```

## `useMedia`

`useMedia` is a custom hook that returns the breakpoints object with boolean value if your current screen size is a particular breakpoint or not. It is also responsive to window resizing and returning the appropriate value according to the new window size.

```tsx
import { useMedia } from '../../';
```

```tsx
const MediaExample = () => {
  const media = useMedia();

  return (
    <View
      style={{
        flexDirection: media.lg ? 'row' : 'column',
        gap: 10,
      }}
    >
      <StyledBox>
        <StyledText>Universal</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Performant</StyledText>
      </StyledBox>
      <StyledBox>
        <StyledText>Accessible</StyledText>
      </StyledBox>
    </View>
  );
};
```

## `useColorMode`

`useColorMode` is a custom hook which return current color mode.

```tsx
import { useColorMode } from '../../';
```

Let's use our `useColorMode` value to give background color to `View` from `react-native`.

```tsx
import { View } from 'react-native';
import { useColorMode } from '../../';

function Example() {
  const colorMode = useColorMode();
  return (
    <View
      style={{
        width: 100,
        height: 100,
        backgroundColor: colorMode === 'light' ? 'white' : 'black',
      }}
    />
  );
}
```
