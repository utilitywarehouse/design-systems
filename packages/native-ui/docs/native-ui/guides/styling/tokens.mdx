# Theme Tokens

`native-ui` uses a design token system to provide a consistent and scalable way to manage design properties.

`native-ui` ships with default tokens that provide access to theme values and lets you build and customise your own themed components.
It has colors, typography, size, opacity, shadows, breakpoints and more.

## How to Use Tokens

To use the theme tokens in your components, you can use the `useTheme` hook to access the theme values or pass a function
to `StyleSheet.create` to access the theme values. You can also use the `Box` component and the `$` prefix to access the theme values.

```tsx
import { Text, Box, useTheme } from '@utilitywarehouse/native-ui';

const MyComponent = () => {
  const {
    // You can access the theme values via the useTheme hook
    colors,
    borderRadius,
    space,
  } = useTheme();

  return (
    <Box
      // You can use the theme values from the useTheme hook
      p={space['150']}
      // Or the built in typed values
      m="200"
      bg={colors.grey50}
      // You can use the $ prefix for colours
      borderColor="$green700"
      borderWidth={2}
      borderRadius={borderRadius['lg']}
    >
      <Text style={styles.text}>Hello, world!</Text>
    </Box>
  );
};

const styles = StyleSheet.create(theme => ({
  text: {
    // You can access the theme values via the theme argument
    fontFamily: theme.font.body,
    fontSize: theme.fontSize['200'],
    color: theme.colors.grey900,
  },
}));
```

## Colours

The colour system is based on the `colour-system` package which provides a set of tokens for both light and dark themes.
To use the colours tokens in your components, you can use the colours below for example `color: colors.cyan400;`.

### Light Colours

These colours should be used in the default light `colorMode` and theme.

<ColorPalette>
  {Object.keys(light).map(key => (
    <>
      <ColorItem
        title={`${key}`}
        subtitle="Light Theme"
        colors={Object.keys(light[key])
          .slice(0, 6)
          .reduce((acc, subKey) => {
            acc[subKey] = light[key][subKey].value;
            return acc;
          }, {})}
      />
      <ColorItem
        title={`${key}`}
        subtitle="Light Theme"
        colors={Object.keys(light[key])
          .slice(6, 12)
          .reduce((acc, subKey) => {
            acc[subKey] = light[key][subKey].value;
            return acc;
          }, {})}
      />
      {Object.keys(light[key]).length > 12 && (
        <ColorItem
          title={`${key}`}
          subtitle="Light Theme"
          colors={Object.keys(light[key])
            .slice(12, 18)
            .reduce((acc, subKey) => {
              acc[subKey] = light[key][subKey].value;
              return acc;
            }, {})}
        />
      )}
    </>
  ))}
</ColorPalette>
